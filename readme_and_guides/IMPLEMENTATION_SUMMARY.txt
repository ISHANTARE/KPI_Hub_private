📈 NEW CORE FEATURES IMPLEMENTED

1. Unified SQLite Data Layer
   - The platform now operates on SQLite operational tables (`data/kpihub.db`) 
     for sub-second dashboard performance.
   - Runtime CSV files are dynamically imported using SQLAlchemy.

2. Bcrypt-hashed Login Authorization
   - Interactive login screen gate added to the entry point.
   - Restricts Viewer roles from seeing financial metrics or writing configurations.

3. REST API Service Sidecar
   - Provides programmatic integrations via FastAPI, checking live DB ping health
     and exposing structured JSON objects for active programs.

4. Expanded Automated Test Coverage
   - Unit tests increased from 5 to 110 cases covering rebalancing, 
     EVM formulas, and role validation.

5. Dynamic Simulation Workspace
   - Adds the Scenario Simulation page to project variance calculations.

📊 SYSTEM OVERVIEW

✅ Multi-Page Streamlit App Architecture (web_app.py + pages/)
   - Fully restructured from a flat tabbed UI into 9 distinct pages.
   - Clean navigation sidebar controlled dynamically by role-based access.

✅ SQLite Database Layer (data/kpihub.db)
   - Powered by SQLAlchemy models and dataLoader logic.
   - Automatically synchronizes CSV flat files into clean SQL tables on startup 
     and when editing/uploading files.

✅ Bcrypt Session Authentication (config/users.yaml + lib/auth.py)
   - Secure login screen on launch.
   - Restricts Manager pages (Integrations, Data Upload) from Viewers.
   - Enforces Viewer data scope access (Viewer sees only assigned project data).

✅ REST API Sidecar (api/main.py)
   - Programmatic access utilizing FastAPI.
   - Live endpoints for monitoring health, retrieving projects, and fetching KPIs.

✅ Automated Sync Scheduler (integrations/scheduler.py)
   - Modular connectors for Jira, Codebeamer, GitHub, SAP, and Outlook.
   - Runs continuously or as a one-shot process (`--once`).

🎯 PAGE INDEX & CORE CAPABILITIES

1. Portfolio Overview  - Unified KPI Engine (Maturity, OTD, EVM bar charts) and alerts.
2. Project Health      - Project timeline tracking, ECR changes, and risk heatmaps.
3. Dev Operations      - GitHub commit stats, pull request velocity, and build success rates.
4. Testing & Quality   - Verification test cases, traceability metrics, and gap analysis.
5. Resource allocation - Engineering workload charts and AI-driven rebalancing recommendations.
6. System Integrations - Configurations UI, credentials check ("Test Auth"), and data pulls.
7. Data Upload         - File uploader to write CSV files and re-sync SQL database.
8. AI Insights         - Standard compliance search and insights engine.
9. Scenario Simulation - Tweak FTEs or ASIL levels to model variance on budget and schedules.

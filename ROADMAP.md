# KPI Hub — Platform Integration Roadmap

This document outlines the current active integration connectors in KPI Hub, as well as planned integration modules for enterprise extensions.

---

## Active Connectors (Core 7)

| Connector | Status | Purpose |
|-----------|--------|---------|
| **Jira** | Active | Issue, Epic, Sprint, and Ticket synchronization |
| **Codebeamer** | Active | Automotive ALM requirements, design reviews, and test cases |
| **Email (SMTP/IMAP)** | Active | Daily digest reports, escalation alerts, and automated notifications |
| **Microsoft Teams** | Active | Webhook alerts, channel cards, and instant team notifications |
| **GitHub** | Active | Development metrics, repository commits, and PR cycle times |
| **Slack** | Active | Real-time chat alert notifications and incident webhooks |
| **Outlook Calendar** | Active | Milestone deadline synchronization and event tracking |

---

## Planned / On-Demand Enterprise Modules

These connectors can be activated or developed based on customer requirements:

- [ ] **Azure DevOps**: Work item tracking, CI/CD pipeline status, and test runs
- [ ] **IBM DOORS / Polarion**: Legacy requirements management sync
- [ ] **Confluence**: Automated documentation export and wiki sync
- [ ] **SharePoint**: Document repository and file artifact synchronization
- [ ] **Power BI**: Direct BI dataset export and embedding
- [ ] **TestRail / Xray**: Dedicated QA test execution management
- [ ] **Asana / Monday.com**: Light project task board synchronization
